# Project Summary

This Visitor Pass Management System digitizes the old visitor register process.

## Roles

- Admin manages users, dashboard, reports, and all data.
- Security creates passes and scans visitors in/out.
- Employee invites visitors and approves/rejects appointments.
- Visitor can register and be included in appointment/pass flow.

## Collections

- Users
- Visitors
- Appointments
- Passes
- CheckLogs

## Flow

1. Visitor details are saved.
2. Employee creates or approves an appointment.
3. Security issues a QR pass.
4. Front desk scans QR for check-in.
5. Same pass can be scanned for check-out.
6. Dashboard and CSV export show reports.
